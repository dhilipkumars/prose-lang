# Hello World CLI Examples

This directory contains simple Command Line Interface (CLI) applications generated in Go. They serve as minimal examples of using `prose-lang` to build standalone binaries that print formatted text.

**Compiled From:** `namaste.prose`  
**Generated Stack:** Go  

## Getting Started

To run the generated application:

```bash
cd generated/namaste
go run main.go
```

**Example Output:**
```
Namaste World, 2026-02-23T22:53:24Z
```
