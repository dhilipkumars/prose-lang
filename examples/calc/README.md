# Mandarin Calculator

A simple calculator single-page application built with React that uses Mandarin numerals. It accepts standard numerical input logic but displays numerals in Mandarin script (e.g., 零, 一, 二 ... 九).

**Compiled From:** `src/calc.prose`  
**Generated Stack:** React (Create React App/Vite), CSS

## Getting Started

To run the generated application:

```bash
cd generated/calc
npm install
npm run dev
```
